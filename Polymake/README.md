The file `n6r4_homology.poly` contains a [Polymake](https://polymake.org) script to compute the homology of the Vietoris-Rips of complex at scale 4 of the hypercube $Q_6$ with the Hamming distance. To run this script, download the file to your computer, then run `polymake --script n6r4_homology.poly` from the same folder. **Warning:** we estimate this computation requires at least 800 GB of RAM. The script can be customized to change the size of the hypercube and the scale of the VR complex.
